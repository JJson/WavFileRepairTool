## Installation

WavFileRepairTool is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'WavFileRepairTool', :git => 'https://github.com/JJson/WavFileRepairTool.git'
```
